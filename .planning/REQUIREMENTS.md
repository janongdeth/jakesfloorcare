# Requirements: Jake's Floor Care Website

**Defined:** 2026-04-16
**Core Value:** Every page looks like one cohesive brand, ranks for local floor care searches, and drives calls/quotes through GHL.

## v1 Requirements

### Design Consistency

- [ ] **DESIGN-01**: All inner pages use homepage's Sora/Outfit fonts and blue (#2a4592) / dark (#141414) color palette
- [ ] **DESIGN-02**: All inner pages use homepage's component patterns (section-tag, section-title, buttons, cards, reveal animations)
- [ ] **DESIGN-03**: Consistent header/nav/footer across all pages matching homepage exactly
- [ ] **DESIGN-04**: No AI-looking design elements (no gradient blobs, glassmorphism, generic icon grids, rounded-2xl, emoji icons)
- [ ] **DESIGN-05**: Mobile sticky CTA bar on all pages
- [ ] **DESIGN-06**: Mobile tap highlights disabled on interactive cards, image long-press disabled on content images

### Page Rebuilds

- [ ] **PAGE-01**: Rebuild 11 service pages to match homepage design language
- [ ] **PAGE-02**: Rebuild 10 service area pages to match homepage design language
- [ ] **PAGE-03**: Rebuild About page to match homepage design language
- [ ] **PAGE-04**: Rebuild Reviews page to match homepage design language
- [ ] **PAGE-05**: Rebuild Contact page with GHL form embed, matching homepage design
- [ ] **PAGE-06**: GHL chat widget integrated site-wide
- [ ] **PAGE-07**: Select and process best photos from client asset folder for each page use case

### SEO

- [ ] **SEO-01**: Technical SEO audit and fixes (meta tags, canonicals, heading hierarchy, crawlability)
- [ ] **SEO-02**: Local SEO optimization (NAP consistency, LocalBusiness schema on all pages)
- [ ] **SEO-03**: Schema markup on all pages (Service, FAQ, Review, BreadcrumbList, etc.)
- [ ] **SEO-04**: XML sitemap generation
- [ ] **SEO-05**: Image SEO (descriptive alt tags, .webp format, lazy loading, compression)
- [ ] **SEO-06**: On-page content optimization for target keywords per service/area page
- [ ] **SEO-07**: Geo-targeted content optimization on service area pages

### AEO

- [ ] **AEO-01**: FAQ sections with concise, AI-snippet-friendly answers on service pages
- [ ] **AEO-02**: Search experience optimization (clear headings, structured answers, featured snippet formatting)

### Performance

- [ ] **PERF-01**: Core Web Vitals optimization (LCP, CLS, FID)
- [ ] **PERF-02**: All images converted to .webp, quality 80, max 1200px wide

## v2 Requirements

### Future Additions

- **BLOG-01**: Blog or content marketing pages for ongoing SEO
- **GALLERY-01**: Project gallery with before/after photos
- **REVIEW-01**: Live Google review feed integration

## Out of Scope

| Feature | Reason |
|---------|--------|
| Redesigning the homepage | Homepage is the approved master design |
| New pages beyond existing set | Scope is fix what's there, not add new |
| Backend or CMS | Stays static HTML |
| Blog/content pages | Deferred to v2 |
| Mobile app | Web only |
